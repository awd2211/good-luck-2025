## 🔧 技术实现

### 核心组件

**Drawer（抽屉）**:
```tsx
<Drawer
  title={模块标题}
  placement="right"
  onClose={关闭处理}
  open={是否显示}
  width="80%"
>
  {模块内容}
</Drawer>
```

**卡片网格**:
```tsx
<Row gutter={[16, 16]}>
  {items.map(item => (
    <Col xs={24} sm={12} md={8} lg={6}>
      <Card hoverable onClick={() => openModule(item.key)}>
        {/* 卡片内容 */}
      </Card>
    </Col>
  ))}
</Row>
```

### 状态管理
```typescript
const [drawerVisible, setDrawerVisible] = useState(false)
const [selectedModule, setSelectedModule] = useState<string | null>(null)

const openModule = (key: string) => {
  setSelectedModule(key)
  setDrawerVisible(true)
}
```

## 💡 使用场景

### 适合场景
- ✅ 功能模块较多（10+）
- ✅ 各模块相对独立
- ✅ 需要频繁切换查看
- ✅ 希望首页简洁美观
- ✅ 需要良好的移动端体验

### 不适合场景
- ❌ 功能模块很少（<5个）
- ❌ 需要同时查看多个模块
- ❌ 用户熟悉程度要求极高
- ❌ 需要快速连续操作

---

## 🎓 用户操作指南

### 查看功能
1. 浏览主页面，找到需要的功能卡片
2. 点击卡片，右侧弹出抽屉
3. 在抽屉中查看和操作
4. 完成后关闭抽屉，返回主页面

### 切换功能
1. 关闭当前抽屉
2. 点击其他功能卡片
3. 或：在抽屉中完成操作后直接关闭

### 快捷操作
- **ESC键**: 快速关闭抽屉
- **点击遮罩**: 关闭抽屉返回主页
- **日期选择**: 在顶部修改，应用到所有模块
