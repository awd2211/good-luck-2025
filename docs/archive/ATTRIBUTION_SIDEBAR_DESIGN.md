### 视觉特点

**左侧图标栏**:
- **宽度**: 60px（固定）
- **背景色**: #001529（深蓝黑色，Ant Design默认暗色主题）
- **图标大小**: 20px
- **图标间距**: 4px
- **图标容器**: 48x48px，圆角4px

**图标状态**:
- **未选中**: 半透明白色 `rgba(255, 255, 255, 0.65)`
- **选中**: 蓝色背景 `#1890ff` + 纯白色图标
- **悬停**: 浅灰背景 `rgba(255, 255, 255, 0.1)` + 纯白色图标
- **过渡动画**: 0.3s 平滑过渡

**顶部标题**:
- 显示"归因统计系统 | 当前模块名称"
- 动态更新当前模块名

## ✨ 交互特性

### 1. 图标导航
- **点击**: 切换到对应模块，内容区域立即更新
- **悬停**: 显示Tooltip提示模块名称
- **选中**: 图标高亮显示（蓝色背景）
- **平滑过渡**: 所有状态变化都有0.3s动画

### 2. 内容切换
- **无抽屉**: 内容直接在右侧区域切换，无需弹出/收起
- **即时响应**: 点击图标后立即渲染新内容
- **保持状态**: 日期范围等全局状态在切换时保持

### 3. 空间优化
- **最小侧边栏**: 只占用60px宽度
- **最大内容区**: 右侧内容区域最大化利用空间
- **垂直滚动**: 图标栏支持滚动（当模块很多时）

## 🔧 技术实现

### 核心代码

**侧边栏配置**:
```typescript
const sidebarItems = [
  {
    key: 'overview',
    icon: <DashboardOutlined />,
    label: '数据概览',
    tooltip: '数据概览',
  },
  // ... 其他13个模块
]
```

**图标栏渲染**:
```tsx
<Sider width={60} style={{ background: '#001529' }}>
  {sidebarItems.map(item => (
    <Tooltip title={item.tooltip} placement="right">
      <div
        onClick={() => setSelectedKey(item.key)}
        style={{
          width: '48px',
          height: '48px',
          background: selectedKey === item.key ? '#1890ff' : 'transparent',
          color: selectedKey === item.key ? '#fff' : 'rgba(255, 255, 255, 0.65)',
          // ... 其他样式
        }}
      >
        {item.icon}
      </div>
    </Tooltip>
  ))}
</Sider>
```

**内容切换**:
```typescript
const renderContent = () => {
  switch (selectedKey) {
    case 'overview': return <OverviewTab dateRange={dateRange} />
    case 'events': return <ConversionEventsTab />
    // ... 其他13个组件
  }
}
```

## 🎓 使用说明

## 🔄 对比其他设计

### vs 两级Tab
- **图标栏**: 更简洁，空间利用率更高
- **两级Tab**: 分组更明确，适合初次使用

### vs 卡片式仪表板
- **图标栏**: 适合频繁切换的专业用户
- **卡片式**: 适合偶尔查看的管理者

### vs 手风琴折叠
- **图标栏**: 适合独立查看各模块
- **手风琴**: 适合对比查看多个模块
